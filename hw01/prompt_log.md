# AI 辅助编程交互日志

### 1. 需求描述
我向 AI 描述了需要解决“八皇后问题”，并要求按照标准 Python 工程结构（src/tests）进行代码生成。要求算法使用回溯法，并包含 N=4 和 N=8 的单元测试。

### 2. Bug 处理与自主修复
* **手动引入 Bug**：我尝试将 `solve_n_queens` 中的递归终止条件从 `row == n` 改为 `row == n + 1`。
* **AI 修复过程**：运行后发现程序无法返回任何结果。我将代码发给 AI 观察，AI 准确指出递归深度超出了棋盘行数，导致无法触发结果收集逻辑。根据 AI 建议改回 `row == n` 后，测试通过。

### 3. 代码重构引导
我引导 AI 优化了数据结构，将原本的列表查找（List）改为集合查找（Set），以提升在大规模 N 值的搜索效率。
