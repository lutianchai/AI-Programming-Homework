# AI 辅助编程实践：八皇后问题求解器 (HW01)

本项目是《人工智能导论》课程的第一次作业，旨在通过 AI 辅助工具完成从工程构建、逻辑实现到自动化测试的全流程开发。

## 1. 项目说明
本项目实现了一个基于**回溯算法**的 N 皇后问题求解器，支持自定义棋盘规模 $N$。

### 核心功能：
* **高效搜索**：利用回溯算法寻找所有合法解。
* **状态优化**：在搜索过程中使用集合（Set）记录攻击路径，将冲突检测的时间复杂度优化至 $O(1)$。
* **工程规范**：遵循标准 Python 项目目录结构，逻辑与测试分离。
